npm i .  -  установить сборку. (в текущую папку)

npm run dev  -  Режим разработки (в этом режиме папку release можно показывать дизайнеру)
npm run build  -  преобразовать в окончательный вариант (для передачи программисту)
npm run addfonts  -  Перевести шрифты в формат woff, woff2 (если есть папка fonts). Для подключения шрифтов нужно удалить файл style/scss/fonts.scss (если такой есть)
